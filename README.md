# csci2720_project

### Backend Setup
1. Navigate to the backend folder
```
cd csci2720_project/backend
```
2. Install backend dependencies:
```
npm install
```

### Frontend Setup
1. Navigate to the frontend folder:
```
cd csci2720_project/frontend
```
2. Install frontend dependencies:
```
npm install
```

### Starting the Application
1. Start MongoDB
Ensure your local MongoDB server is running. By default, the app connects to:
```
mongodb://localhost:27017/projectDB
```
2. Start Backend Server (inside /backend)
```
node server.js
```
3. Start Frontend (inside /frontend)
```
npm start
```

### Site URLs
```python
http://localhost:3000 # (or the port shown in the frontend terminal)
```
